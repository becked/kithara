# Kithara

A cross-platform soundboard application for Old World game audio.

Built with [Tauri 2](https://tauri.app/), [Svelte 5](https://svelte.dev/), and Rust.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://rustup.rs/) (stable)
- [Old World](https://store.steampowered.com/app/597180/Old_World/) game installed

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```
